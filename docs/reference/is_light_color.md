# Check if a color is light

Check if a color is light

## Usage

``` r
is_light_color(color)
```

## Arguments

- color:

  A hex color string or color name

## Value

Logical indicating if the color is light

## Examples

``` r
is_light_color("#FFFFFF")
#> [1] TRUE
is_light_color("#000000")
#> [1] FALSE
```
