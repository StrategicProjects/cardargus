# Wrap text to fit within a given width

Wrap text to fit within a given width

## Usage

``` r
wrap_text(text, max_width, font = "Jost", fontsize = 12)
```

## Arguments

- text:

  Text to wrap

- max_width:

  Maximum width in pixels

- font:

  Font name

- fontsize:

  Font size

## Value

List with wrapped lines and total height
