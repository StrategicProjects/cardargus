# Download a Google Font as WOFF2 to the cache

Download a Google Font as WOFF2 to the cache

## Usage

``` r
download_google_font_woff2(family, weight = "400")
```

## Arguments

- family:

  Font family name.

- weight:

  Numeric/character weight (default "400").

## Value

Path to cached WOFF2 file or NA_character\_.
